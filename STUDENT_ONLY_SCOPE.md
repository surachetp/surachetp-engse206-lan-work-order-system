# Student-only Repository Scope

Repository นี้ตั้งใจให้มีเฉพาะสิ่งที่นักศึกษาต้องใช้ทำและส่งงาน

## มีใน repository นี้

- คู่มือเริ่มต้นและคู่มือ Git
- Pre-learning
- Activity guide และ template P00–P20
- โฟลเดอร์ evidence, assets, references และ submission
- AI usage log, team information และ progress dashboard

## ไม่มีใน repository นี้

- `grading/`
- `teaching-materials/`
- `release/`
- rubric ภายในของผู้สอน
- slide และ teaching notes ของผู้สอน
- model solution / เฉลย Campus Resource Booking
- instructor role cards หรือ feedback comment bank
