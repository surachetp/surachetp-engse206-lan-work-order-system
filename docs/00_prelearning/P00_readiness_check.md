# P00 — Readiness Check

ตอบสั้น ๆ ด้วยคำของตนเองก่อน Day 1

1. **Problem** ต่างจาก **requirement** อย่างไรในกรณีของคุณ
2. ยกตัวอย่าง **functional requirement** 1 ข้อที่อาจเกิดในกรณีของคุณ
3. ยกตัวอย่าง **non-functional requirement** 1 ข้อที่อาจสำคัญ
4. ใครคือ stakeholder ที่อาจมีเป้าหมายขัดกัน และเรื่องใด
5. สิ่งใดที่คุณยังไม่รู้และต้องเก็บ evidence เพิ่ม
6. ถ้ากรณีนี้ขยายใหญ่เกินไป คุณจะตัดอะไรออกก่อน
