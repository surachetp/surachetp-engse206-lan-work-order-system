# P19 — Design Pitch

## Pitch Storyline (8–10 minutes)

1. Problem and stakeholders
2. Evidence and insights
3. Prioritized requirements / NFR
4. Design strategy and architecture
5. User flow / prototype
6. Detailed design of one critical workflow
7. Review evidence, revisions and remaining risks
